# sample1

## 概要

- 悪いマークアップの例。
- 見た目は整っているが、すべてdivタグで実現しており、見出しタグやランドマークタグを使っていない。

## ファイル構成

- index.html
- index.css
- another.html

## index.htmlのページ構成

- ヘッダー
- サイドメニュー（４つのメニューはすべてonclickイベントでanother.htmlに遷移する）
  - 新着情報
  - 人気記事
  - おすすめ記事
  - お知らせ
- メインコンテンツ
  - 新着情報一覧を表示する。
  - 一覧の各アイテムは以下の要素からなる。
    - 記事のサムネイル
    - 記事のタイトル
    - 記事の説明文
